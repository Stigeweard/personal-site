AlgoDuel

A platform to challenge your programmer friends to coding duels.

wow!

This was my capstone project at Galvanize. My goal was to learn a new backend language in two weeks and implement advanced features such as a database connection, password hashing, socket connections, and custom made tests for code that was written in a text editor on the front end.

I decided on Python and Flask for my server because of it's quick prototyping and then used various Flask plugins to achieve my desired goals. It is currently not hosted because of security issues associated with allowing users to submit any code they want that is then interpreted by the server.

Future work on this project will focus on sanitizing the users input so that I can have it hosted and challenge people to (coding) duels!

TwitchVis

Data visualization project. Twitch is a platform to stream video game tournaments and all viewers simultaneously have access to the chat room. This results in large tournaments having a largely unintelligible chat room. TwitchVis remedies this problem by creating a visual representation of the most commonly used words from the past 30 seconds, updating every few seconds.

See it in action! (modal video)

View on GitHub!

Features
  • ChartJS populating with commonly said words
  • Stop words such as "and, like, if, or" filtered out
  • Twitch OAuth
  • Users can chat in chatroom
  • User can load different stream
